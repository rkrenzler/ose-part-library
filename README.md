# OSE part library

This workbench helps to have fast a access to different FreeCAD files.


## Installation
[See](https://www.freecadweb.org/wiki/How_to_install_additional_workbenches)

### Linux

````
$ mkdir ~/.FreeCAD/Mod
$ cd ~/.FreeCAD/Mod
$ git clone https://github.com/rkrenzler/ose-part-library.git
````
